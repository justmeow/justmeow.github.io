macOS Catalina 10.15.x U盘重装

2020-05-24 记录 mac 评论

虽然mac电脑可以直接初始化电脑，但是通过自带的重装系统，默认装的版本一般都比较低，花很多时间装好后还要花很多时间升到最新系统。所以推荐使用U盘重装。

提示

  • 提前准备一个8G以上的可覆盖U盘
  • 仅适用与mac电脑重装，黑苹果绕道
  • 官方镜像不提供直接下载，所以资源难寻，千万别通过任何渠道付费购买镜像
    1. 完全没必要浪费钱。只要有mac电脑就可以自己提取
    2. 8成是通过百度云。那感人的速度，真的是闻者伤心听者落泪

步骤

下载系统镜像

在App Store中搜索系统名称 macOS Catalina(其他版本也一样) 点击获取，就可以看到系统开始下载

U盘准备

接下来的操作会导致U盘格式化，如有重要文件请提前备份

  • 打开 “应用程序 → 实用工具 → 磁盘工具”
  • 将U盘「抹掉」(格式化) 成「Mac OS X 扩展（日志式）」格式、GUID 分区图，并将 U 盘命名为「Catalina」。 注意：这个盘符名称必须与后面的命令里的名称一致，需要认真看清楚，很多新手容易出错在这里。

制作启动盘

  • 在终端中执行:
    sudo /Applications/Install\ macOS\ Catalina.app/Contents/Resources/createinstallmedia --volume /Volumes/Catalina /Applications/Install\ macOS\ Catalina.app --nointeraction
    注意：这个盘符名称必须与后面的命令里的名称一致，需要认真看清楚，很多新手容易出错在这里。

  • 终端中显示如下内容表示制作完成:

    Install media now available at “/Volumes/Install macOS Catalina”

重装系统

  • 重启电脑、按住option键、选择刚才制作的系统安装盘启动
  • 选择「磁盘工具」，将目标磁盘「抹掉」成「APFS」格式，命名为「Macintosh HD」
  • 选择「重新安装macOS」，后续就安装提示操作便可

牢骚

  不得不庆幸自己当初选择了8G内存以及设备的型号(2012年中款)刚好够上了macOS Catalina硬件的末班车，而且还算能够比较流畅的运行。
  不过目前的最大问题就是开始时候的128G小硬盘真的是承受不住了，由于本来容量就不大剩余容量就更少了，导致读写性能十分的让人窒息(也有可能是用的时间太长，擦写次数没剩多少的原因)。所以使得十分想更换掉目前这个硬盘。可惜神秘力量的原因今年固态的价格又突然暴涨，真的后悔当初怎么就不先屯块便宜硬盘。
